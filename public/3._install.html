package.json file

All npm packages contain a file, usually in the project root, called package. json - this file holds various metadata relevant to the project. It is used to give information to npm that allows it to identify the project as well as handle the project's dependencies. All dependencies installed will be listed in the package.json file.

This file can be created with the usual $ touch package.json command or by using $ npm init
