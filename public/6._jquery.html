What's jQuery?

jQuery is at its core, a Document Object Model (DOM) manipulation library. The DOM is a tree structure representation of all elements of a web page. jQuery simplifies the syntax for finding, selecting and manipulating these DOM elements. jQuery can be used for finding and element in the document with a certain property, changing one of more of its attributes, or making it respond to an event (e.g mouse click).

One of the principles of developing with jQuery is the separation of Javascript code from HTML markup. The jQuery library provides a simple syntax for adding event handlers to the DOM using Javascript, rather than adding HTML event attributes.

In this website a jQuery script is used to expand the main content view of the Web pages. If the user presses the (...) sign located in the header it will hide the left and right bars (using toggle method, which checks the visibility) upon mouse click (the click event is sent to the HTML element, when the mouse button is pressed and released). The code is written inside script tags and can be kept in the HTML file, usually at the bottom or as an alternative written in a separate .js file and reference it from the HTML.
