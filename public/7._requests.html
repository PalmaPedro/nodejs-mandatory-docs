Using Postman

Postman is a tool used to test API's making it easier to make Http requests (among other request methods) and it is needed for making POST and PUT requests.

Before start using Postman, however, there is a need to include Express's middleware to deal with the (incoming) data (object) in the body of the request

On the top of the app.js file, the following middleware must be included (below the express import and instantiation). These middleware are functions that are called between processing the Request and sending the Response in the application method. It is only needed for POST and PUT requests, because in both these requests, data is being sent (in the form of some data object) to the server, and it is asked from the server to accept or store this data (object) which is enclosed in the request body( i.e req.body) of that POST or PUT request.
