What's npm?

When node.js is installed, npm, which stands for Node Package Manager is also included. Npm is basically a tool to help install packages needed for a particular project.

npm global or local packages?

An important concept to understand when installing npm packages is to choose wether the installations will be made available local or globally. With npm, there are some global installs, but mostly everything will be done on a local project basis, meaning that it is required to install everything for each project in its own directory.

Any global installs will have the -g flag.
