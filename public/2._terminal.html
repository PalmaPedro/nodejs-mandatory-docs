Using Bash in Windows

Before moving on to the section of this website where it is explained how to set up Node, i think it might be usefull to spend some time learning or refreshing the basics of the terminal commands. As a Windows user myself i was, until this point, not very familiar with the terminal as i have always used Windows Gui. Learning command line tools for managing the file system, can give more control and speed over the workflow. At the present time i am using Bash, but Powershell is also a popular choice.

Bash commands

I will go through some of the most often used commands:

  • pwd - printing working directory
    • $ pwd
  • ls - list directory contents
    • $ ls
  • cd - change directory
    • $ cd folder_name
  • cd .. - go back one directory
    • $ cd ..
  • mkdir - make new directory
    • $ mkdir folder_name
  • touch - make new file
    • $ touch file_name
  • mv - move directory to a new location
    • $ mv [option] [source destination]
  • cp - copy directory to a new location
    • $ cp [option] [source destination]

Git commands

I will also include the most often used git commands while working on a project

  • init - initialize a git repository to a new or existing project
    • $ git init
  • clone
    • $ git clone [repository source]
  • add
    • $ git add .
  • commit
    • $ git commit -m "commit message"
  • push
    • $ git push
  • pull
    • $ git pull
