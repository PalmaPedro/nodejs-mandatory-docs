Using Bash in Windows

Before moving on to the section of this website where it is explained how to set up Node, i think it might be usefull to spend some time learning or refreshing the basics of the terminal commands. As a Windows user myself i was, until this point, not very familiar with the terminal as i have always used Windows Gui. Learning command line tools for managing the file system, can give more control and speed over the workflow. At the present time i am using Bash, but Powershell is also a popular choice.

Bash commands

I will go through some of the most often used commands:

  • mkdir
